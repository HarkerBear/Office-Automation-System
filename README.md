# Office-Automation-System
## Introduction:
Hello this is an office automation system which automates leave application and audit. The technologies I used in this system are:
* Backend: Java+MyBatis 	
* Frontend: FreeMarker+LayUi
* Dependencies: Maven 		
* Test: JUnit+Logback
* Access Control: RBAC		
* Data Protection: MD5
